# גורי! 🍌 Minions Stadium

A single shareable web page: a full-screen crowd of CSS-only minions jumping in a
stadium and chanting **"גורי!"** in layered human voices every 2.2 seconds.

**Live:** https://chenbenari.github.io/Morning_Routine/

## How it works

- `index.html` — the whole experience (CSS minions, start overlay, chant engine).
  Web Audio API layers 7 randomized voices per beat (volume 0.65–0.9, playback
  rate 0.97–1.03, stereo pan ±0.5, 0–150ms delay) over a bandpass-filtered crowd
  noise loop, with an HTMLAudio fallback. Audio only starts after the user taps
  the start button (browser autoplay policy).
- `audio/gori-play-1.mp3 … 12.mp3` — Hebrew voice clips synthesized with Edge TTS
  (`he-IL-HilaNeural` / `he-IL-AvriNeural`, elongated + rate/pitch-varied).
  Generated by `generate_audio.py`; the deploy workflow creates them on the CI
  runner if missing and commits them back (the dev sandbox has no TTS egress).
- `og-image.svg` — social sharing preview.
- `test-audio.mjs` — Playwright smoke test: `npm install playwright && node test-audio.mjs`

## Deploy

`.github/workflows/deploy-gori-minions.yml` publishes this folder to GitHub Pages
on every push that touches it.
